# Second Brain AI - Flutter Application

## Overview
Second Brain AI is a personal productivity and task management application built with Flutter, designed to act as a "second brain" for users. The application leverages AI-based suggestions to help users optimize their tasks and time usage. This project demonstrates the integration of AI insights, local task management, and scheduled notifications, making it a highly useful tool for students, professionals, and anyone seeking structured task organization.

## Key Features
- **User Authentication**: Users can sign in with name, email, and password.  
- **Task Management**: Add multiple tasks with different timestamps and priorities.  
- **AI Suggestions**: Generates intelligent suggestions for tasks based on context from the backend API.  
- **Notifications**: Users can schedule notifications for important tasks and AI suggestions.  
- **Task History**: Maintains a persistent history of tasks and suggestions until the user chooses to delete them.  
- **Logout Functionality**: Securely log out while preserving task history.

## Project Significance
This project serves as a comprehensive demonstration of:
1. **Flutter Cross-Platform Development** – a single codebase for Android, iOS, and web.  
2. **AI Integration** – connecting to a running backend AI API to deliver actionable suggestions.  
3. **Task Automation** – scheduling notifications for improved productivity.  
4. **State Management and Persistence** – storing user data and task history reliably on the device.  

The project highlights the practical application of AI in daily task management and demonstrates how AI can augment human productivity effectively.
